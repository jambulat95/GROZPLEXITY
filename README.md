# Grozplexity AI — Реверс-инжиниринг контента для блогеров

## Какую задачу решает проект

**Grozplexity AI** — это платформа для автоматического анализа и реверс-инжиниринга вирусного контента. Система помогает блогерам и контент-мейкерам:

- **Понять, почему их видео становятся вирусными** — анализирует визуальный стиль, структуру, аудио и выявляет "вирусные триггеры"
- **Создавать контент в узнаваемом стиле** — генерирует новые сценарии на основе "ДНК стиля" автора
- **Преодолеть творческий кризис** — предлагает готовые идеи и структуры для новых видео
- **Масштабировать производство контента** — автоматизирует процесс создания сценариев

### Технические возможности

- **Мультимодальный AI-анализ**: анализ видео, аудио и текста с помощью Gemini 2.5 Flash
- **Извлечение метаданных**: автоматический сбор статистики (просмотры, лайки, комментарии, платформа)
- **Создание "Style Passport"**: детальный разбор визуального стиля, темпа, хуков, структуры
- **Генерация сценариев**: создание новых сценариев в стиле автора на любую тему
- **Синтез Master Profile**: объединение всех анализов в единую "ДНК стиля" автора

## Результаты в цифрах

- **Время анализа видео**: ~2-3 минуты (включая загрузку, транскрибацию и AI-анализ)
- **Точность определения вирусных триггеров**: анализ на основе реальных метрик (просмотры >100K)
- **Поддержка платформ**: YouTube Shorts, Instagram Reels, TikTok, Facebook
- **Оптимизация запросов**: обработка 3 ключевых кадров (512x512) для быстрого анализа
- **База данных**: SQLite с хранением истории анализов и профилей авторов

## Откуда появилась идея

Идея проекта родилась из реального общения с блогерами и контент-мейкерами. В процессе работы с ними я заметил повторяющуюся проблему: даже успешные авторы, чьи видео набирают миллионы просмотров, часто не понимают, **почему именно этот контент стал вирусным** и как воспроизвести этот успех.

После серии интервью с блогерами разных ниш выяснилось, что:

- **Творческое выгорание** — авторы не знают, о чем снимать дальше
- **Непонимание своего стиля** — сложно описать, что делает их контент узнаваемым
- **Ручной анализ** — тратят часы на разбор чужих видео, пытаясь понять структуру
- **Нет системы** — каждый раз изобретают велосипед, не используя накопленный опыт

Осознав масштаб проблемы, я решил создать инструмент, который автоматизирует процесс реверс-инжиниринга контента. Вместо того чтобы вручную анализировать десятки видео, блогеры могут загрузить ссылку и получить:

- Детальный разбор стиля с конкретными рекомендациями
- Готовые сценарии для новых видео
- Понимание своих "вирусных триггеров"
- Систематизацию накопленного опыта

Проект решает реальную боль блогеров и помогает им масштабировать производство качественного контента.
